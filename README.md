# Chatzy – Real-Time Chat Platform

Chatzy is a backend-focused real-time chat application built using **Django** and **Django REST Framework** with **JWT authentication** and **PostgreSQL** database.  

The project is designed as a learning-oriented yet production-style system that demonstrates how modern backend services are structured, secured, and deployed.

---

## Features

- Custom User Model  
- User Registration API  
- JWT Login & Token Refresh  
- Secure Password Hashing  
- PostgreSQL Database Integration  
- Django Admin Panel  
- RESTful API Architecture  
- Modular App Structure  

**Planned Features**

- Real-time messaging using WebSockets  
- Private & group chats  
- Online / Offline status  
- File & image sharing  
- Docker & AWS deployment  

---

## Tech Stack

- Python
